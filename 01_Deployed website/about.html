What this is

Daftar is a practice of one. Fixed scope, fixed fee, one practitioner — the work is done by the person you spoke to, not handed down to a team you never meet. Each engagement is priced and bounded before it begins, and ends in a written deliverable plus the editable working file.

This is not an audit firm. Daftar does not issue audit opinions, sign financial statements, or provide any attest service. The work is non-attest advisory only — technical review, statement preparation, audit readiness, and transaction support — kept deliberately separate from the assurance function so it never compromises an auditor's independence or its own.

What you get is senior judgement on the specific technical question in front of you, without the overhead, the handoffs, or the lock-in. The file is yours; the position is one you can defend.
